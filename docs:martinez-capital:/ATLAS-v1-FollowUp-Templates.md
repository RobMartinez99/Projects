# ATLAS v1 — FOLLOW-UP TEMPLATES
## Martinez Capital | Post-Call Communication

**Usage rules:**
- All templates are DRAFT-ONLY. Agent reviews and personalizes before sending.
- Do not send any template that references a specific carrier, rate, or health detail without verifying it first.
- Tone: warm, non-pushy, family-first. No "today only" urgency. No guilt-tripping.
- Replace all [brackets] before sending.
- These are not legally reviewed compliance documents. Agent is responsible for FL compliance on all outbound communications.

## CATEGORY 1 — POST-CALL SMS (Called, Spoke, Did Not Close)

### Template SMS-1A — General Follow-Up (No Decision)
```
Hi [First Name], this is [Agent Name] with Martinez Capital. 
Great speaking with you today. I've got a couple of options 
set aside that fit your budget and situation. Just reply 
when you're ready and I'll walk you through them. 
No pressure — I'm here when you need me. — [Agent Name]
```

### Template SMS-1B — Beneficiary Mentioned (Personalized Follow-Up)
```
Hi [First Name], [Agent Name] here. Wanted to make sure 
[Beneficiary Name] has something in place if anything 
happens. I have a plan that fits your budget — just say 
the word and I'll pull it up for you. I'm just a text away.
```

### Template SMS-1C — "Thinking It Over" Follow-Up (24–48 hrs)
```
Hey [First Name], [Agent Name] from Martinez Capital. 
Just checking in — wanted to make sure you had 
everything you need to feel good about this. 
Happy to answer any questions. No rush. — [Agent Name]
```

### Template SMS-1D — "Spoke With Spouse/Family" Follow-Up
```
Hi [First Name], [Agent Name] here. Hope you had a 
chance to chat with [spouse/family member]. I'm happy 
to get on a quick call with both of you if that helps.
```

## USAGE NOTES FOR ATLAS

- Atlas can suggest the appropriate template based on call outcome.
- Atlas will not auto-send any template — agent sends all communications.
- If a prospect asked a specific health or carrier question on the call, Atlas will draft a custom answer insert for EMAIL-4B rather than leaving the [Answer] brackets blank.
- All template content is draft-only. Agent is responsible for accuracy and FL compliance before sending.
- No rate figures, carrier names, or health class references should appear in any template unless the agent adds them manually and has verified accuracy.
