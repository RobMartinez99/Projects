## CARRIER #1 — AMERICO (Eagle Premier / Eagle Guaranteed)
**Source:** Americo Eagle Premier.pdf + Know_Your_Carriers.webp

**Rob's directive:** Always run first.

### Eagle Premier (Level Benefit)
| Field | Detail |
|---|---|
| Issue Ages | 40–85 (Nonsmoker) / 40–80 (Smoker) |
| Face Amount | $5,000–$40,000 |
| Annual Fee | $40/year (commissionable) |
| Benefit Tier | **Level — full benefit from day one** |
| Rate Classes | Nonsmoker / Smoker · Male / Female |
| Underwriting | Instant eApp · Rx check + MIB review |

**Health routing — binary decision (from Americo agent guide):**
- All health questions (except smoker) = **NO** + within build chart → **Eagle Premier issued**
- Any health question = **YES** OR outside build chart → **routes to Eagle Guaranteed (GI)**

> ⚠ Americo uses a binary system, not a 3-tier Select structure. Rob's "Select 1/2/3" maps to: Nonsmoker rates (best), Smoker rates (higher), Eagle Guaranteed (waiting period). Not separate tiers within Eagle Premier.

**Key riders:**
- Accidental Death Benefit (ADB)
- Accelerated Benefit Rider (chronic/terminal illness)
- Child/Grandchild Rider available

**Smoking/Nicotine — Quit Smoking Advantage (unique feature):**
- Smokers qualify for **nonsmoker rates for 3 years**
- At year 3: if still smoking → face amount reduces OR client pays higher premium
- If quit by year 3 → keeps nonsmoker rates at full face amount
- This is a major competitive advantage for smoker clients

**Rob's practical notes:**
- Good bonus structure
- Great redraft policy (easy to correct billing issues)
- Good tech with alerts
- Quick payouts

## CARRIER #5 — AMERICAN-AMICABLE (Golden Solution / Senior Choice)
**Source:** American-Amicable_Golden_Solution_Plans_WL_UnderwritingGuide.pdf + Senior Choice Agent Guide.pdf + Know_Your_Carriers.webp

**Rob's directive:** Competitive rates, lenient UW. Not great on the back end. Bad redraft policy.

> **Note:** Two Am-Am FE product lines exist in the folder. Golden Solution and Senior Choice are distinct products with slightly different face amount maximums. Both serve ages 50-85. Use the Golden Solution guide for primary reference.

### Tier Structure (Both Products — 3 Tiers)

**TIER 1 — Immediate Death Benefit (Level)**
- Condition: All Questions 1–8 answered NO
- Benefit: Full face amount from day one

**TIER 2 — Graded Death Benefit**
- Condition: Question 8 answered YES (Qs 1–7 = No)
- Year 1: 30% of face amount
- Year 2: 70% of face amount
- Year 3+: 100% full face amount

**TIER 3 — Return of Premium Benefit**
- Condition: Questions 4–7 answered YES (Qs 1–3 = No, Q8 = No)
- Year 1–2 or 1–3: Return of all premiums + 10% interest
- After graded period: 100% full face amount

**Knockout — NO coverage if Questions 1–3 = YES** (terminal illness, institutionalized care, or similar knockouts — reference guide for exact question wording)

### Golden Solution Face Amounts
| Ages | Immediate Max | Graded / ROP Max | Minimum |
|---|---|---|---|
| 50–75 | $50,000 | $25,000 | $2,500 |
| 76–85 | $25,000 | $25,000 | $2,500 |

### Senior Choice Face Amounts
| Tier | Max | Minimum |
|---|---|---|
| Immediate | $35,000 | $2,500 |
| Graded or ROP | $20,000 | $2,500 |

**Annual fee:** $30/year (both products)
**Monthly modal factor:** 0.088

**Social Security billing codes (Senior Choice guide):**
- 1S, 3S = 1st and 3rd of month (SS deposit dates)
- 2W, 3W, 4W = 2nd, 3rd, 4th Wednesday

**Available riders (Golden Solution):**
- Grandchild Rider: $5,000/unit, up to 2 units, $12/year per grandchild
- Nursing Home Waiver of Premium Rider

**Rob's practical notes:**
- Competitive rates
- Lenient underwriting — helpful for clients with more health conditions
- Long underwriting process (many decisions, slower than Americo/Corebridge)
- Outdated technology
- Bad redraft policy — be careful with billing setup

## QUICK DECISION GUIDE FOR ATLAS

```
CLIENT PROFILE                          → SUGGESTED CARRIER PATH

Healthy — all health Qs = No            → Americo Eagle Premier (Nonsmoker or Smoker rate class)
  + within build chart                     Backup: Mutual of Omaha Level

Any health Q = Yes                      → Americo Eagle Guaranteed (GI, ages 50–80)
  OR outside build chart                   Backup: Am-Am Graded / ROP
                                           Backup: Corebridge SimpliNow Legacy (Graded)

Smoker / recent tobacco use             → Americo Eagle Premier (Quit Smoking Advantage)
                                           NS rates for years 1–3; quit by yr 3 = NS for life

Multiple conditions, unclear routing    → Run Corebridge first to see what the
                                           system flags, then route accordingly

Cannot qualify Level anywhere           → Americo Eagle Guaranteed (GI, ages 50-80)
                                           or Am-Am Graded / ROP

Denied by all standard carriers         → ETHOS Advantage Whole Life (last resort)
                                           ⚠ Verify with Rob — no agent guide on file
```
